# 📊 Demand Forecasting using XGBoost in R

## 📌 Project Overview

This project focuses on predicting product demand using a machine learning model built with XGBoost in R. The goal is to analyze historical data and generate accurate demand forecasts to support better decision-making.

---

## ⚙️ Tools & Technologies

* R
* XGBoost
* dplyr
* ggplot2
* caret

---

## 🔍 Workflow

1. Data loading and exploration
2. Data preprocessing and cleaning
3. Train-test split (80-20)
4. Feature preparation (dummy variables)
5. Model training using XGBoost
6. Prediction on test data
7. Model evaluation using RMSE
8. Visualization of actual vs predicted demand

---

## 📈 Model Details

* Algorithm: XGBoost (Regression)
* Objective Function: `reg:squarederror`
* Evaluation Metric: RMSE

---

## 📊 Results

The model predicts demand values and compares them with actual values. Performance is evaluated using RMSE and visualized using scatter plots and line charts.
